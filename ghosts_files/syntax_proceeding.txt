# SYNTAX OF GENERAL CODING
## Project Name + Version
e.g.: myfempy v0.1.0

version control {X => current version of the project/ big changes require a new version 
		 Y => update performance in the current version*
		 Z => fix bugs and instabilities in the current version

X.Y.ZaN   # Alpha release
X.Y.ZbN   # Beta release
X.Y.ZrcN  # Release Candidate
X.Y.Z     # Final release

*add new function or modules will change this sub version


# Set Name class/ methods... 
folders name		=>	abc

class  name		=>	AbcAbc/
methods 'def'  name	=>	abc()/ setAbcAbc/ getAbcAbc/ newAbcAbc/ addAbcAbc/ runAbcAbc ...
variable name 		=>	a{single low}/ A{single upp}/ abc{short name}/ abc_abc{long name}
